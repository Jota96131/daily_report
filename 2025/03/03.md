## 分かったこと

　 • git branch `<ブランチ名>`: ブランチを新規追加する

　 • git branch -a: ブランチの一覧を表示する。

　 • git checkout `<既存ブランチ名>` or git switch: ブランチを切り替える

　 • git branch -m　<ブランチ名>: ブランチ名の変更。

　 • git branch -d　<ブランチ名>: ブランチを削除する。
